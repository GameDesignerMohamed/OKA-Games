Dead Beat
March 15, 2026
Conditional
Game
Top-down rhythm brawler across 3 arenas (100/120/140 BPM escalation). A shrinking ring indicates the beat window — on-beat punch lands 3× damage with freeze-frame and chromatic aberration; off-beat lands 1×. On-beat dodge grants full iframes; off-beat dodge is movement only. Two enemy types: Grunts (1-beat telegraph) and Heavies (2-beat telegraph). Boss per arena. Beat-timing governs offense and defense simultaneously from the same clock.
Design
The on-beat/off-beat distinction holds as a genuine mechanic class distinct from Pulse. Pulse was timing-as-accuracy; Dead Beat is timing-as-risk-management — the same clock produces two simultaneous outputs (attack multiplier + iframe window). Dodge-offense coupling is the real skill ceiling: a player who masters the beat is safer AND deadlier, which creates compounding advantage without feel-bad punishes. Primary gap: perfectHits and maxCombo are tracked in code but not persisted — player improvement is invisible across runs. No cross-run mastery loop means the skill ceiling exists but the player can't see themselves climbing it.
Verdict
The unified attack/defense timing clock is a genuinely novel substrate — same clock, two simultaneous outputs, compounding advantage for mastery without double-punishment for failure. Three conditions to reach PURSUE: (1) per-arena localStorage personal best (Perfect Hits % + Max Combo), (2) boss power-beat pattern every 4th beat for rhythm-within-rhythm, (3) mid-arena BPM ramp +5 per wave. The one thing that would make it sing: an anti-beat enemy type that attacks on the off-beat — forces players to suppress their conditioned response, which is the hardest and most interesting skill ceiling test with no new art required.

Conditional
Game
Click-to-assign agentic RTS. You command 4 autonomous agents (Scout, Builder, Fighter, Medic) by clicking goals — they execute autonomously, with hard role rejection when a goal is outside their spec: "NOT MY JOB" / "HEAL ONLY" / "TOO CLOSE." Design
Select-assign-watch does feel like meaningful direction, not delayed mouse control — the combination of click-to-assign plus autonomous idle behavior is what creates the director feeling. Rejection bubbles make agent constraints legible in real-time — clearest agent constraint communication in any OKA-Games build to date. The fix is a live situation report ticker per agent showing current autonomous intent (one line each: "SCOUT → engaging rusher-3", "MEDIC → healing Fighter 2HP"). That single addition transforms the player from reactive corrector into proactive director — and puts this in a category no commercial title has shipped. Scout confirmed: "agent reasoning transparency as first-class mechanic" is 0% shipped in the market.

Clearance
March 13, 2026
Conditional
Game
Top-down wave survival where your Clearance Level (0→10) gates which mechanics exist. Start with move + pistol only. Every 2 waves, one ability unlocks mid-run with a full-screen flash and audio sting: Dash at CL2, Grenade at CL4, Assault Rifle at CL6, Shield parry at CL8, Extraction at CL10. Enemy density and speed scale each wave. Reach CL10 and extract to win. Personal best tracked across runs.
Design
Progressive authority is a distinct mechanic class not previously tested in the OKA portfolio. The constraint window (pistol-only waves 1–2) establishes the skill floor so the CL2 Dash lands as a reward, not a tutorial. The unlock sequence has correct behavioral ordering: Dash solves positioning → Grenade solves density → Rifle solves DPS → Shield solves survival. Each unlock answers a problem the previous stage created. Enemy scaling across three speed tiers (CL0–3 / CL4–7 / CL8–10) is legible and telegraphed. Core gap: the unlock sequence is linear with no player agency — run 2 is structurally identical to run 1, and per-ability usage is invisible (CL personal best tracks how far, not how you played).
Verdict
Earned access thesis confirmed. The first Dash unlock after 2 pistol-only waves is the proof: constraint → reward → investment. The CL bar creates visible anticipation, telegraph rings solve fairness, ability cooldown bars solve visibility, tiered audio stings build ascending power feeling. The 5 unlocks aren't equal — Dash and Grenade change how you play; Rifle is a fire-rate bump, Shield is a weaker iframe. Fix list: Rifle needs a dramatic fire-pattern change (burst/pierce/tracer), Shield needs a true parry window (≤0.15s), grenade charges refill on wave clear, enemy speed tiers need visual distinction (color/size), BGM should layer with each CL unlock, end screen should show ability unlock timeline and per-ability usage.

But the game is currently trivially winnable: credit drain is too slow and there's no passive attrition, so the optimal strategy is to delete nothing and wait. Fix the economy to force 4–5 deletions per run and add passive loyalty decay. Then every deletion is a real loss you chose to take. The one thing that would make it sing: make the pressure match the emotional core.

Game
Top-down puzzle game on a conveyor belt. Packages with three visible properties (COLOR × SIZE × MATERIAL) travel into an AI scanner with a hidden boolean rule drawn from a pool of 12. Submit harmless probe packages to observe what gets flagged — probes never cost HP, only smuggled contraband does. Build a mental model, exploit the gap you find, then slip contraband through. Level clear reveals the rule with an aha-moment screen. 5 levels, rule redrawn each run. UnrealBloom scanner glow, camera lerp toward scanner during each scan, Web Audio tension loop.
Design
The probe-for-free model is the core design insight: separating probe cost (zero HP) from contraband cost (HP loss) makes deduction feel like skilled investigation rather than guessing. Verdict
The player feels like a detective who gets smarter each run — not because RNG was kind, but because they learned to read the AI. Probe-free costs vs. contraband costs creates a genuine skill ceiling: disciplined inference outperforms brute-forcing every time, and the player feels that gap close as they improve. The competitive category is essentially empty — Zachtronics is the closest neighbor, but turn-based and static; Cipher is real-time and reactive. The one thing that would make it sing: escalating variable count per level (3 → 4 → 5 properties), so the skill ceiling is explicit and the player's inference model has room to compound.

Pulse
March 6, 2026
Failed
Game
Rhythm bullet-hell in Three.js. The world pulses to a 100 BPM procedural beat — concentric ring obstacles expand outward every beat, each with a gap opening the player must dodge through. Mouse to move, click or spacebar to shoot. On-beat shots fire a cyan pulse with 2× damage and a shockwave ring. Off-beat shots fizzle. 5 escalating waves (BPM rises +5/wave, ring layers increase) plus a boss fight with circular orbit, 3 attack patterns, and random mid-fight tempo jumps. Beat streak of 5 unlocks a synth layer in the music.
Design
Beat sync creates a unique dual-skill demand: spatial dodging AND rhythm timing simultaneously. The ring gap forces the player to read beat timing spatially — you can't just dodge randomly, you must be at the gap when the ring hits. On/off-beat feedback is explicit: "ON BEAT!" vs "OFF BEAT" flash (Rule #3 — no silent failure). Wave clear has a 1s zoom-exhale breathing room before next wave (Rule #7). Boss tempo jumps (±10–20 BPM) are the mechanical peak — the player's beat window suddenly shifts, demanding re-sync under pressure. Combo layer (music gains a synth channel at 5× streak) is the standout feel moment: the music rewards you audibly.
Verdict
Rhythm mechanic class is real (two-variable constraint: position + timing), but the prototype has three named structural gaps, not one: (1) Gap marker — gap position is invisible, making dodge a spatial search not a timing skill. (2) Near-beat grace zone — 0.5× at 50ms; current binary 0×/1× is luck not reaction. (3) Combo multiplier scoring — no accumulation reward for maintaining rhythm. Three fixes required. Shoot cooldown misalignment (0.28s allows 2 shots/beat — only first counts rhythmically) and combo resets on damage = double punishment. Rebuild with all three fixes in one cycle and re-evaluate — mechanic class is worth pursuing.

Game

Conditional
Game

Collect 5 data shards scattered across the maze, then reach the exit portal. Avoid the Sentinel: a red adaptive AI that patrols your previous routes using a heatmap. The wall-clipping bug breaks the emergence model and must be fixed before this is worth scaling to multi-Sentinel. One fix: fix diagonal wall-clipping and add Sentinel hearing (footstep radius) — speed vs silence vs route variance as three independent axes would make every run feel strategically distinct.
