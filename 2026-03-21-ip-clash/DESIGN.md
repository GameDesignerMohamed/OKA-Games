# IP Clash

**Concept:** A real-time arena shooter where 3 fictional IP cards each bend the rules of combat in a distinct way — split shots, time slow, or enemy chaos — testing whether named IP branding creates a meaningfully different power fantasy from generic power-ups.

**Player fantasy:** "I am a franchise owner deploying my IP's signature move — and everyone knows it when I do."

## Core Loop
Move and shoot against 3 waves of spawning enemies. At any point, play one of 3 one-use IP cards to dramatically shift the game's physics for 8 seconds (giant title flash, rule-break, tension return). Survive wave 3 to win.

## Key Mechanics
- **IP Activation:** Full-screen title card + color palette shift signals ownership moment
- **Rule-bending cards:** Each IP changes a *different* game rule (projectile spread vs. time vs. enemy behavior)
- **One-use constraint:** Irreversibility makes the activation moment meaningful — no spamming, real decisions

## What's Built
Prototype proves: IP branding on activation does measurable psychological work even with box-sprite graphics. Neon Syndicate (enemies home toward each other) is the standout — genuine emergent chaos the player takes credit for. Named > unnamed is confirmed at this scope. Card timing strategy emerged naturally without being designed in.
